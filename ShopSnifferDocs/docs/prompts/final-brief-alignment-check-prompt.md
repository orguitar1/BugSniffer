# Final Alignment Check Prompt

Use this prompt for a quick alignment pass after all agents have reviewed a plan, design document, or major decision. Paste into each agent to confirm everyone is on the same page before proceeding.

---

We are about to move forward with implementing the agreed plan. Before we begin, confirm the following:

1. **Agreement** — Do you agree with the plan as stated? If not, what specific concerns remain?

2. **Gaps** — Is there anything the plan doesn't address that you think should be resolved before implementation starts? Be specific — cite the section or decision that's missing or incomplete.

3. **Risks** — What's the single biggest risk you see in this plan? Not a general concern, but a specific scenario where something could go wrong during implementation.

4. **Dependencies** — Are there any external dependencies (API access, tool availability, design decisions from other components) that could block implementation? Flag them now.

If you have no concerns, state that explicitly and confirm you're ready to proceed. Do not give a vague "looks good" — state what you've verified and why you're confident.
